Spring Store 🍃

Simple online store on the Spring framework (Java). In particular, for this project I used Spring Boot, MVC (model-view-controller), Thymeleaf template engine and Spring Security for authentication and access-control. Also, I took Hibernate ORM as a Java Persistence API realization. I used PostgreSQL as a database for this project and Git version control system. My Spring Store application deployed to Heroku cloud application platform.
